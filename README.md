# README #

### What is this repository for? ###
* This our repository for UTD Hackathon.

### How do I set up? ###

* Get the latest repo
> git clone https://bitbucket.org/vadivel_selvaraj/comet-parking.git
> cd comet-parking

* Install NPM dependencies
> npm install

* Start the server
> npm start
